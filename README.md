# Date Countdown Website

A stylish countdown timer website showing the days passed and remaining between March 26, 2026 and September 21, 2026.

## Features

- **Real-time Countdown**: Displays days passed, days remaining, and progress percentage
- **Star Wars Theme**: Custom Star Wars Empire propaganda background
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Auto-updating**: Progress bar and stats update every minute
- **Modern UI**: Beautiful gradient cards and smooth animations

## Files

- `index.html` - Main webpage structure
- `style.css` - Styling and layout
- `script.js` - Countdown logic and calculations

## How to Use

1. Open `index.html` in your web browser
2. The countdown will automatically calculate and display:
   - Days that have passed since March 26, 2026
   - Days remaining until September 21, 2026
   - Progress percentage
   - Current date

## Customization

You can modify:
- Start and end dates in `script.js`
- Colors and styling in `style.css`
- Background image URL in `style.css`

## Live Preview

Simply open the `index.html` file in any modern web browser to view the countdown.
